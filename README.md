# Pixel Positions – Job Portal (Laravel)

Pixel Positions is a job portal web application built using Laravel where users can browse featured jobs, search by role, and employers can post job listings.

## 🚀 Features
- User authentication (Login & Logout)
- Job listing & featured jobs section
- Search jobs by role
- Employer job posting functionality
- Clean dark-themed UI
- Responsive design

## 🛠️ Tech Stack
- Laravel 12
- PHP
- MySQL
- HTML, CSS, JavaScript

## ⚙️ Installation Guide

1. Clone the repository:
git clone https://github.com/dev-akashshar/pixal_positions_laravelProject_Job.git

2. Go to project directory:
cd pixal_positions_laravelProject_Job

3. Install dependencies:
composer install

4. Copy environment file:
cp .env.example .env

5. Generate app key:
php artisan key:generate

6. Configure database in `.env` file:
DB_DATABASE=your_database_name  
DB_USERNAME=root  
DB_PASSWORD=

7. Run migrations:
php artisan migrate

8. Start the server:
php artisan serve
